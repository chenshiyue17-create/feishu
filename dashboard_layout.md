# 小红书飞书仪表盘设计

目标是把当前这批“5 个账号、156 条作品”的小红书数据，在飞书多维表格里搭成接近参考图那种一屏式运营看板。

## 当前数据基线

最近一次同步可直接支撑这些核心指标：

- 监控账号数：`5`
- 总粉丝数：`1,937`
- 总获赞收藏数：`16,282`
- 总作品数：`156`
- 总点赞数：`895`
- 平均点赞数：`5.74`
- 视频作品数：`57`
- 图文作品数：`99`
- 视频占比：`36.54%`
- 当前头部作品：`老板G30玫瑰盐系‼️上海内购🔥不允许买贵了`
- 当前头部作品点赞：`94`

## 数据源分工

- `小红书仪表盘总控`
  用于顶部 KPI 卡和全局说明。
- `小红书看板总览`
  用于账号对比、账号排行、账号结构分析。
- `小红书看板榜单`
  用于作品排行榜、TOP3 卡片、爆文清单。
- `小红书看板趋势`
  用于账号级时间趋势。
- `小红书作品数据`
  用于明细钻取、筛选、二次分析。

## 推荐布局

### 1. 顶部指标卡

数据源：`小红书仪表盘总控`

建议放 6 张卡，顺序按参考图调整为：

- `监控账号数`
- `总粉丝数`
- `总获赞收藏数`
- `总作品数`
- `总点赞数`
- `平均点赞数`

可选扩展两张：

- `视频作品数`
- `视频占比`

### 2. 左侧账号排行榜

数据源：`小红书看板总览`

建议做 2 个横向条形图，二选一放左侧，另一个放底部：

- 图表 A：`账号粉丝排行`
  - 维度：`账号`
  - 指标：`粉丝数`
  - 排序：`粉丝数` 降序
- 图表 B：`账号总点赞排行`
  - 维度：`账号`
  - 指标：`首页总点赞`
  - 排序：`首页总点赞` 降序

### 3. 中间 TOP3 内容卡片

数据源：`小红书看板榜单`

过滤条件：

- `卡片标签` 属于 `TOP1`、`TOP2`、`TOP3`

卡片展示建议：

- 主图：`封面图`
- 主标题：`标题文案`
- 副标题：`作品摘要`
- 辅助信息：`账号`
- 点击跳转：`作品链接`

当前这批数据的 TOP3 可以直接对应：

- `老板G30玫瑰盐系‼️上海内购🔥不允许买贵了`
- `老板 K 系列油烟机咋选？看完不纠结✨`
- `老板R1P➕U1P2️⃣种安装效果展示‼️`

### 4. 底部双图

建议做成两张柱线组合图。

图 1：账号内容体量与互动

- 数据源：`小红书看板总览`
- X 轴：`账号`
- 柱：`首页总点赞`
- 线：`首页平均点赞`

图 2：账号内容结构

- 数据源：`小红书看板总览`
- X 轴：`账号`
- 柱：`首页视频数`
- 柱：`首页图文数`
- 线：`视频占比`

如果你更想看时间变化，就把图 2 换成：

- 数据源：`小红书看板趋势`
- X 轴：`数据更新时间`
- 柱：`首页总点赞`
- 线：`首页平均点赞`
- 筛选器：`账号`

## 飞书仪表盘组件建议

### 组件顺序

第一行：

- 6 张指标卡

第二行：

- 左 1/3：账号排行
- 右 2/3：TOP3 内容卡

第三行：

- 左 1/2：账号互动图
- 右 1/2：账号结构图或趋势图

第四行：

- 明细表：`小红书作品数据`

### 视觉方向

- 主题建议选深色
- 指标卡统一用白字 + 强对比数字
- 柱图主色用白或浅灰
- 线图强调色用红色或洋红
- TOP3 卡片保持 16:9 封面比例

## 操作建议

1. 先把 `小红书仪表盘总控` 放成顶部指标卡。
2. 再用 `小红书看板榜单` 的 `TOP1~TOP3` 做中间卡片区。
3. 左侧排行和底部柱线图统一用 `小红书看板总览`，这样结构最稳。
4. 趋势图只在你后续积累 7 天以上快照后再放大，否则信息量不够。
